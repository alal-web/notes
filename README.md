# Notes

Менеджер заметок с поддержкой Markdown, тегов, папок и полнотекстового поиска. Данные хранятся в SQLite, веб-интерфейс открывается в браузере на десктопе или мобильном устройстве.

---

## Установка и запуск

```bash
git clone https://github.com/alal-web/notes
cd notes
```

### Требования:
 установленный Go (версия 1.21+)
### Установка инструментов сборки
```
go install github.com/air-verse/air@latest
go install github.com/evanw/esbuild/cmd/esbuild@latest  
```
Добавьте $HOME/go/bin в PATH, если ещё не сделали этого.

```
# Сборка и запуск
make
make dev
```
Приложение будет доступно по адресу: http://localhost:8080

## Возможности
### Папки (директории)
Организуйте заметки по папкам. Можно создавать, переименовывать и удалять папки, перемещать заметки между ними.

### Markdown
Каждая заметка пишется в формате Markdown. Поддерживаются заголовки, списки, ссылки, изображения, таблицы и другие элементы.

### Теги
Присваивайте заметкам теги для быстрой фильтрации. Одна заметка может иметь несколько тегов.

### Полнотекстовый поиск
Поиск работает по заголовку, содержимому и тегам заметок. Результаты сортируются по релевантности (ранжирование BM25).

## Хранение данных
Все заметки, папки и теги сохраняются в локальной базе данных SQLite (файл note.db в директории проекта). Это означает, что:

* Данные не теряются при перезапуске приложения

* Приложение работает без интернета (offline-first)

* Базу данных можно скопировать или перенести на другое устройство

## Технологии
* Backend: Go (Golang)

* База данных: SQLite

* Frontend: HTML, JavaScript, CSS (ESBuild для сборки)

* Поиск: FTS5 (Full-Text Search) SQLite с ранжированием BM25
